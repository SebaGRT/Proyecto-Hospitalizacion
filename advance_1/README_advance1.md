# Advance 1 — Carga y Exploración Inicial

## Objetivo
Carga eficiente de ~5.8M registros GRD 2019–2024, exploración inicial, diseño metodológico y definición de hipótesis.

## Archivos
- `Analisis_Hospitalizacion.ipynb` — Análisis exploratorio inicial
- `Proyecto-Hospitales.ipynb` — Notebook principal de avance 1
- `EDA_Egresos_Hospitalarios_Chile.ipynb` — EDA de egresos hospitalarios

## Logros
- Carga optimizada de datos GRD públicos (separador `|`, encoding UTF-8)
- Exploración de distribuciones de estancia, procedimientos y diagnósticos
- Definición de grupos diagnósticos: Neoplasias (C50, C18-C20, C53, C34) y Sepsis (A40, A41)
- Diseño de hipótesis H1, H2, H3
- Identificación de variables clave para modelado
